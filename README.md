🛒 Supermarket Data Analysis Project
📌 Overview

This project is a data analysis of a supermarket dataset.
The goal is to explore the data, clean it, and extract meaningful insights that can help understand customer behavior, sales trends, and business performance.

This project was built as part of my learning journey in Data Analysis using Python.

🎯 Objectives
Understand the structure of the dataset
Perform data cleaning and preprocessing
Handle missing values and duplicates
Analyze sales and customer behavior
Extract useful business insights
🧰 Tools & Technologies
Python 🐍
Pandas
NumPy
Matplotlib
Jupyter Notebook
📊 Project Steps
1. Data Loading
Importing the dataset into Pandas DataFrame
Checking basic structure using head(), info(), and describe()
2. Data Cleaning
Handling missing values
Removing duplicates
Checking and fixing data types
3. Exploratory Data Analysis (EDA)
Analyzing sales patterns
Understanding customer types
Exploring product categories
Detecting trends over time
4. Visualization
Bar charts for sales comparison
Distribution plots
Trend analysis graphs
🔍 Key Insights
Identified top-performing product categories
Observed customer purchasing patterns
Found sales trends across different segments
Highlighted data quality issues and how they were fixed
